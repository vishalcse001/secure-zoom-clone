# 📹 Secure Zoom Clone

A secure, real-time video conferencing application inspired by Zoom. This project allows users to seamlessly connect through high-quality video and audio calls.

## ✨ Features
* Real-time Video and Audio Calling
* Secure Meetings and Rooms
* Mute/Unmute Audio & Turn On/Off Video

## 💻 Tech Stack
* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Real-time Communication:** WebRTC, Socket.io

## 🚀 How to Run Locally

### 1. Setup Server (Backend)
\`\`\`bash
cd server
npm install
npm start
\`\`\`

### 2. Setup Frontend
\`\`\`bash
cd frontend
npm install
npm start
\`\`\`
